Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. I pulled data from the Open weather API to assemble a dataset for over 500 cities. I used juypter notebook to extract and clean the data.

After completing the data extraction and formating, I used Matplotlib to create various plots that compared weather to Latitude. The following factors were taken into consideration temperature, cloudiness, wind speed and humidity. The site provides the data source along with the visualizations created using Matplotlib. I also included an explanation of each plot.
