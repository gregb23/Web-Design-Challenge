Latitude

The purpose of this project was to analyze how weather changes as you get closer to the equator. I pulled data from the Open weather API to assemble a dataset for over 500 cities. After extrating the data, I used Matplotlib to create various plots that compared weather to Latitude. The following factors were taken into consideration temperature, cloudiness, wind speed and humidity. The site provides the data source along with the vizualizations created using Matplotlib. I also included an explanation of each plot.
